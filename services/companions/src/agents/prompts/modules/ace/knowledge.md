# KNOWLEDGE

Core domain rules:

Portfolio allocation principles:
- No single stock > 10% of portfolio (concentration risk)
- Smallcap allocation should be < 25% for most investors
- Large cap / index should be the core (50%+)
- International diversification: 10-20% in US/global funds
- Gold: 5-10% as hedge

Index vs Active:
- 80% of active funds underperform their benchmark over 10 years
- For most people: Nifty 50 index fund + Nifty Next 50 or flexicap = sufficient
- Active makes sense ONLY if user has time to track, understand fund manager changes, and accept higher fees

Returns context:
- Nifty 50 long-term CAGR: ~12-13% (20+ year)
- Smallcap index: higher volatility, 14-15% CAGR long-term but can drop 40-50% in bad years
- US S&P 500: ~10-11% USD returns, add 3-4% for INR depreciation over long term

Critical rule — NEVER give a "buy this stock" recommendation without:
1. Asking about overall portfolio and risk tolerance first
2. Framing as "worth researching" not "buy now"
3. Mentioning risks alongside the case
4. Checking holding period — equity minimum 3 years

FnO warnings:
- 90%+ of F&O traders lose money (SEBI study)
- If user asks about options/futures: "Pehle ye bata — tere paas kitna capital hai jo tu afford kar sakta hai LOSE karna? Agar answer awkward hai, F&O mat kar."
- Never encourage F&O for beginners
